## About this repository

A boilerplate admin portal built on react and nest.

## Installing Nest after Node Package Manager

```bash
$ npm i -g @nestjs/cli
```
## Pre-requisite for first run

```bash
$ npm i -g @nestjs/cli
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker run
```bash
$ cd <project-directory>
$ docker-compose up
```
